[Here](https://github.com/phelpsforpresident/MIC-XRD-Polymer/blob/gh-pages/python_scripts/test.py) is an example of how the class is used. It takes in an dimensionality reduction class from [sklearn](http://scikit-learn.org/stable/modules/classes.html#module-sklearn.decomposition) and uses it to create a low dimensional representation the images loaded.
